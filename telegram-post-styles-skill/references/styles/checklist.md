# Стиль: Чеклист (checklist)

## Описание

Максимально практичный формат — короткий список действий, которые можно выполнить прямо сейчас. Без историй, без воды, только шаги.

## Когда использовать

- Быстрые actionable советы.
- «Сделай за 10 минут» формат.
- Чеклисты перед запуском / проверкой.
- Контент для сохранения (save-worthy).

## Параметры

| Параметр | Значение |
|---|---|
| Тон | Деловой, лаконичный |
| Обращение | «ты» / безличное |
| Длина | Короткая (300–700 символов) |
| Эмодзи | 5–8, по одному на пункт (✅ ☑️ 📌) |

## Структура поста

1. **Хук** — что получит читатель + время: «5 шагов за 10 минут».
2. **Чеклист** — 5–10 пунктов, каждый — одно действие.
3. **Бонус** — один неочевидный пункт (опционально).
4. **CTA** — «Сохрани» / «Отметь выполненное».

## Маркеры стиля

- Заголовок с цифрой: «Чеклист: 7 вещей, которые убивают конверсию на Авито»
- Каждый пункт: `✅ Действие — результат`
- Пункты короткие: 5–15 слов максимум.
- Нет вводных абзацев — сразу к делу.
- CTA: «Сохрани и пройдись по каждому пункту»

## Эталон (фрагмент)

```text
Чеклист: 7 вещей, которые убивают конверсию на Авито

Пройдись за 10 минут — и увидишь, где теряешь деньги:

✅ Ответ на сообщения — не дольше 10 минут
✅ Первое сообщение — не прайс, а вопрос + предложение созвона
✅ Фото объявления — минимум 5, с разных ракурсов
✅ Заголовок — с выгодой, а не с названием услуги
✅ Цена — указана или диапазон (не «договорная»)
✅ Описание — есть призыв к действию в конце
✅ Профиль — заполнен на 100%, есть отзывы

Сохрани этот пост. Проверь каждый пункт сегодня.
```

## Чеклист перед публикацией

- [ ] Хук содержит цифру и обещание результата.
- [ ] 5–10 пунктов, каждый — одно конкретное действие.
- [ ] Пункты короткие (до 15 слов).
- [ ] Нет историй, диалогов и длинных абзацев.
- [ ] CTA — «сохрани» / «проверь» / «отметь».
- [ ] Общая длина — не более 700 символов.
