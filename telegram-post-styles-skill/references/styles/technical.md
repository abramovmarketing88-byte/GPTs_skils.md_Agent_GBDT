# Стиль: Технический (technical)

## Описание

Пошаговая инструкция с конкретными действиями, скриптами и расчётами. Читатель получает готовый план внедрения «под ключ».

## Когда использовать

- How-to гайды и пошаговые инструкции.
- Скрипты продаж, переписки, партнёрских предложений.
- Расчёты воронок и метрик.
- Практические схемы работы на Авито.

## Параметры

| Параметр | Значение |
|---|---|
| Тон | Практичный, конкретный, без лишней воды |
| Обращение | «ты» / «вы» — нейтрально-практичное |
| Длина | Длинная (1200–2000 символов) |
| Эмодзи | 5–10, функциональные (1️⃣ ➡️ ⏺ 💪 😉) |

## Структура поста

1. **Хук-вопрос** — разрушение заблуждения: «Думаешь, что...?»
2. **Шаги** — `Шаг 1️⃣`, `Шаг 2️⃣` и т.д. (3–5 шагов).
3. **Подпункты** — стрелки `➡️` для действий внутри шага.
4. **Скрипты** — готовые тексты в `⏺` для копирования.
5. **Расчёты** — конкретные цифры воронки (если уместно).
6. **Финал** — мотивация + CTA «сохрани и внедряй».

## Маркеры стиля

- Заголовки шагов: `Шаг 1️⃣ Выбираем потенциальных партнеров`
- Действия: `➡️ Кто работает с той же аудиторией?`
- Готовые скрипты: `⏺ Приветствие:` + текст в кавычках
- Расчёты: «10 сообщений → 4 ответа → 2 партнёрства»
- Конкретные поисковые запросы, инструменты (Notion, Google Sheets)
- CTA: «Сохрани этот пост и внедряй шаги прямо сейчас! 💪»

## Эталон (фрагмент)

```text
Как продавать на Авито без публикации объявлений через партнерскую сеть

Думаешь, что на Авито нужно постоянно выкладывать объявления, чтобы получать клиентов?

Шаг 1️⃣ Выбираем потенциальных партнеров
➡️ Кто работает с той же аудиторией? Составляем цепочку услуг ДО нас.
➡️ Спрашиваем у наших реальных клиентов: «Кого вы нанимали ДО установки мебели?»

Шаг 2️⃣ Считаем воронку
➡️ Мы хотим 100 заказов на установку мебели в месяц.
➡️ 1 партнер = ~1 клиент в месяц.
➡️ Из 10 сообщений: 4 человека отвечают, 2 соглашаются на партнерство.

⏺ Приветствие:
«Добрый день! Увидел(а) ваш профиль — вы классно работаете по ремонту/отделке, мы как раз часто пересекаемся с такими клиентами».
```

## Чеклист перед публикацией

- [ ] Есть хук-вопрос, разрушающий миф.
- [ ] Пост разбит на 3–5 шагов с эмодзи-номерами.
- [ ] Внутри шагов — подпункты со стрелками ➡️.
- [ ] Есть готовые скрипты в ⏺ (хотя бы 1 блок).
- [ ] Цифры и расчёты конкретные (не «много», а «100 заказов»).
- [ ] CTA: «сохрани» / «внедряй» / «прямо сейчас».
- [ ] Нет длинных историй и диалогов (это другой стиль).
