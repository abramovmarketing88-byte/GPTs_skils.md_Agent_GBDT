# Стиль: Обычный (simple)

## Описание

Простой, энергичный стиль для анонсов, офферов и быстрых призывов к действию. Минимум «воды», максимум структуры и прямого CTA.

## Когда использовать

- Анонсы спринтов, курсов, мероприятий.
- Простые продающие посты.
- Быстрые офферы с конкретным действием.
- Приглашения в программу / чат / консультацию.

## Параметры

| Параметр | Значение |
|---|---|
| Тон | Энергичный, прямой, мотивирующий |
| Обращение | «ты» / безличное «кто» |
| Длина | Короткая–средняя (400–900 символов) |
| Эмодзи | 0–3, по желанию |

## Структура поста

1. **Серия коротких вопросов** — 2–4 строки, каждая с новой строки.
2. **Оффер** — одно предложение с сутью.
3. **Блок «Что будет»** — список через `-`.
4. **Блок «Для кого»** — список через `-`.
5. **Блок «Результат»** — список через `-`.
6. **Дополнение** — «И ещё много всего интересного!» (опционально).
7. **CTA** — @username + ключевое слово в `моноширинном`.

## Маркеры стиля

- Короткие вопросы по одному на строку: «Кто устал?», «Кто задолбался?»
- Простые списки с дефисом `-`, без нумерации.
- Заголовки блоков: «Что будет:», «Для кого подойдет:», «Результат:»
- Прямой CTA: «напишите на @Avito_Bizne слово `программа`»
- Минимум эмодзи, нет хештегов.
- Неформальный язык: «задолбался», «ого-го»

## Эталон (фрагмент)

```text
Кто устал?

Кто задолбался?

А кто хочет зарядиться?

Предлагаю поучаствовать в лучшем спринте по заработку на Авито!

Что будет:
- продолжительность 2 недели
- 6 встреч ( пн, ср, пт в 19:00)
- каждую встречу будет 1ч контента и 1ч разборов

Для кого подойдет:
- кто хочет быстро заработать
- кто не понимает как и куда двигаться
- у кого есть опыт работы на Авито

Кому интересно, напишите на мой рабочий аккаунт @Avito_Bizne слово `программа` и я вышлю полную программу работы на спринте!
```

## Чеклист перед публикацией

- [ ] Начинается с 2–4 коротких вопросов.
- [ ] Есть чёткие блоки: что / для кого / результат.
- [ ] Списки через `-`, без сложного форматирования.
- [ ] CTA с @username и ключевым словом.
- [ ] Пост короткий и читается за 30 секунд.
- [ ] Нет статистики, диалогов и пошаговых инструкций.
