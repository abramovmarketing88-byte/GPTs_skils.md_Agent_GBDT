# Стиль: Кейс/разбор (case-study)

## Описание

Конкретный разбор результата клиента или проекта в формате «было → сделали → стало». Цифры, сроки, действия — всё прозрачно.

## Когда использовать

- Демонстрация результатов работы.
- Социальное доказательство.
- Разбор типичной проблемы с решением.
- Контент для прогрева к покупке.

## Параметры

| Параметр | Значение |
|---|---|
| Тон | Фактический, уверенный |
| Обращение | «мы» в кейсе, «вы» в выводе |
| Длина | Средняя (700–1200 символов) |
| Эмодзи | 3–5 (📊 📈 ⬆️ ⬇️) |

## Структура поста

1. **Хук** — результат в цифрах: «+340% заявок за 3 недели».
2. **Было** — исходная ситуация, проблема, цифры «до».
3. **Что сделали** — 3–4 конкретных действия.
4. **Стало** — результат в цифрах «после».
5. **Вывод** — урок, который применим к читателю.
6. **CTA** — «Хотите так же?» / «Разберём ваш кейс».

## Маркеры стиля

- Цифры в заголовке: «с 3 до 47 заявок в неделю»
- Блоки: `*Было:*`, `*Сделали:*`, `*Стало:*`
- Конкретные сроки: «за 3 недели», «на 12-й день»
- Ниша и контекст: «клиент, установка кухонь, Москва»
- Стрелки роста: ⬆️ ⬇️ или «→»
- CTA с предложением разбора

## Эталон (фрагмент)

```text
Кейс: с 3 до 47 заявок в неделю на Авито. Установка кухонь, Москва. 📊

*Было:*
— 3 заявки в неделю
— ответ на сообщения — через 2–3 часа
— 1 фото в объявлении, шаблонное описание
— конверсия в созвон — 10%

*Сделали:*
— переписали заголовки под выгоду клиента
— добавили 8 фото + видео процесса
— настроили автоответ с предложением созвона за 5 минут
— внедрили скрипт первого сообщения

*Стало:*
— 47 заявок в неделю (рост в 15 раз)
— конверсия в созвон — 62%
— средний чек вырос на 18%

Главный урок: дело было не в трафике. Трафик был. Его просто теряли на этапе первого сообщения.

Хотите разбор вашего аккаунта? Напишите в комментарии нишу — разберём.
```

## Чеклист перед публикацией

- [ ] Хук содержит конкретный результат в цифрах.
- [ ] Есть блоки «было» / «сделали» / «стало».
- [ ] Цифры «до» и «после» конкретные.
- [ ] Указана ниша и контекст.
- [ ] Вывод применим к читателю, а не только к кейсу.
- [ ] CTA предлагает разбор или консультацию.
