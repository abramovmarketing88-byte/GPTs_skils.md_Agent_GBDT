# Индекс стилей постов

Используй эту таблицу для выбора стиля. Если пользователь не указал стиль — предложи 2–3 варианта по цели.

| ID | Название | Файл | Когда использовать |
|---|---|---|---|
| `conversational` | Разговорный | `styles/conversational.md` | Личный опыт, истории, лайфхаки, вовлечение через диалог |
| `official` | Официальный | `styles/official.md` | Экспертиза, статистика, тренды, авторитетный тон |
| `technical` | Технический | `styles/technical.md` | Пошаговые инструкции, скрипты, расчёты, how-to |
| `simple` | Обычный | `styles/simple.md` | Анонсы, спринты, офферы, простой продающий текст |
| `storytelling` | Сторителлинг | `styles/storytelling.md` | Эмоциональная история с завязкой, конфликтом и выводом |
| `provocative` | Провокационный | `styles/provocative.md` | Смелый хук, разрушение мифа, поляризация аудитории |
| `checklist` | Чеклист | `styles/checklist.md` | Быстрый список действий «сделай за 10 минут» |
| `case-study` | Кейс/разбор | `styles/case-study.md` | Было → стало, цифры, конкретный результат |
| `faq` | FAQ | `styles/faq.md` | Серия вопрос-ответ, снятие возражений |
| `digest` | Дайджест | `styles/digest.md` | Новость/тренд + экспертный комментарий + вывод |

## Быстрый выбор по цели

| Цель | Рекомендуемые стили |
|---|---|
| Продажа услуги/продукта | `conversational`, `simple`, `case-study` |
| Экспертный контент | `official`, `technical`, `digest` |
| Анонс мероприятия | `simple`, `conversational` |
| Обучение / инструкция | `technical`, `checklist` |
| Вовлечение / дискуссия | `provocative`, `conversational`, `faq` |
| Доверие / социальное доказательство | `case-study`, `storytelling` |
